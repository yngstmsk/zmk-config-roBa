このドキュメントは、roBaキーボードの各レイヤーのキー配置をアスキーアートで表示しています。

## レイヤー2: 数字/カーソルレイヤー

```
┌─────┬─────┬─────┬─────┬─────┐                 ┌─────┬─────┬─────┬─────┬─────┐
│ ESC │  <  │  >  │  ↑  │     │                 │  7  │  8  │  9  │  +  │  *  │
├─────┼─────┼─────┼─────┼─────┼─────┐     ┌─────┼─────┼─────┼─────┼─────┼─────┤
│ TAB │  $  │  ←  │     │  →  │     │     │     │  4  │  5  │  6  │  -  │  /  │
├─────┼─────┼─────┼─────┼─────┼─────┤     ├─────┼─────┼─────┼─────┼─────┼─────┤
│半/全 │英数 │かな │  ↓  │  0  │     │     │     │  1  │  2  │  3  │  =  │  |  │
└─────┴─────┴─────┼─────┼─────┼─────┤     ├─────┼─────┼─────┴─────┴─────┴─────┘
                  │かな │ WIN │     │     │Space│Enter│
                  ├─────┼─────┼─────┤     ├─────┼─────┼─────┐
                  │ MO1 │ MO2 │ MO3 │     │     │     │     │
                  └─────┴─────┴─────┘     └─────┴─────┴─────┘
```

**機能:**
- カーソルキー: ↑↓←→
- テンキー: 0-9
- 日本語入力切替: 半/全、英数、かな

**注釈:**
- `<` と `>`: `LS(COMMA)` / `LS(PERIOD)`（`JP_LT` / `JP_GT`）を使用（配列設定に依存せず `<` `>` を出力）
- レイヤー1の `\` は JP_YEN (0x89) を使用して円記号/バックスラッシュを出力

```
┌─────┬─────┬─────┬─────┬─────┐                 ┌─────┬─────┬─────┬─────┬─────┐
│ F1  │ F2  │ F3  │ F4  │ F5  │                 │ F6  │ F7  │ F8  │ F9  │ F10 │
├─────┼─────┼─────┼─────┼─────┼─────┐     ┌─────┼─────┼─────┼─────┼─────┼─────┤

│     │     │     │     │     │     │     │     │DISC │DISC │DISC │DISC │DISC │
│     │     │     │     │     │     │     │     │  0  │  1  │  2  │  3  │  4  │
└─────┴─────┴─────┼─────┼─────┼─────┤     ├─────┼─────┼─────┴─────┴─────┴─────┘

                  ├─────┼─────┼─────┤     ├─────┼─────┼─────┐

                  └─────┴─────┴─────┘     └─────┴─────┴─────┘
```

**機能:**
- ファンクションキー: F1-F10
- Bluetooth制御:
  - BT CLR: 現在のデバイスのペアリング削除
  - BT NXT/PRV: デバイス切替
  - BT 0-4: デバイス選択
  - DISC 0-4: デバイス切断
  - ※ BT ALL CLR はレイヤー6に移動済み

**ロータリーエンコーダー:**
- Ctrl+PageUp / Ctrl+PageDown

## キー略語説明

| 略語 | 意味 |
|------|------|
| MO1-2 | モーメンタリーレイヤー1-2 (押している間だけ有効) |
| LClk | 左クリック |
| RClk | 右クリック |
| MClk | 中クリック |
| BSpc | バックスペース |
| DEL | デリート |
| BT | Bluetooth |
| CLR | クリア |
| NXT | 次 |
| PRV | 前 |
| DISC | 切断 |
| 半/全 | 半角/全角切替 |

## レイヤー構造

```
Layer 0 (デフォルト)
  │
  ├── FN (hold) → Layer 4 (FN: F6/F7/F8/F9)
  ├── MO1 (hold) → Layer 1 (シンボル/編集)
  ├── MO2 (hold) → Layer 2 (数字/カーソル)
  │
  └── コンボ
      └── MO1+MO2 同時押し → Layer 3 (ファンクション/BT) 一時有効化

Layer 5 (システム) - 直接アクセス方法は定義されていません
```
